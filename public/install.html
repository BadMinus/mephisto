Installing Mephisto

Get Mephisto up and running like a pro!

  1. Create a database named mephisto (or one of your choosing).
  2. Open config/database.example.yml and set your database credentials.
  3. Rename database.example.yml to database.yml
  4. Upload the entire mephisto directory to your webserver.
  5. Run rake db:bootstrap from a terminal of some sorts
  6. Login to your administration at http://domain.com/admin with the username: admin and password: test
